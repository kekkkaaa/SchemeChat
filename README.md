# SchemeChat

SchemeChat is an `AI War Room` for running multi-model discussion in real web sessions.
You can let ChatGPT, Gemini, Grok, and other providers debate the same task, then guide the flow toward a usable Markdown-style result instead of a pile of chat transcripts.

## Demo

SchemeChat focuses on coordinated multi-provider discussion in a split workspace.
It works for developer tasks like repo planning, implementation review, and execution breakdown, but it also works for normal users who just want to analyze a problem, generate a plan, or check a result.

## Privacy

- 100% Private, no data is collected
- Your login credentials stay between you and your AI provider

## Features

- **AI War Room flow** - Run multi-round discussion that moves from independent analysis to cross-review and final synthesis
- **Task-oriented entry** - Choose whether this round is for analyzing a problem, generating a plan, or reviewing an output
- **Discussion intensity presets** - Quick / Standard / Deep map to 3 / 4 / 5 round orchestration
- **Real web sessions, no API keys** - Use official provider web apps directly and keep provider auth in the provider pages
- **Split workspace** - Keep multiple AI providers visible at once and coordinate them in one workspace
- **MCP connection for external agents** - Codex and Claude Code can read discussion state, update the flow, and trigger the built-in controller
- **Temporary chats for scratch work** - Open temporary/private web chats before exploratory prompts to reduce history pollution
- **Session persistence** - Stay logged in across app restarts
- **Cross-platform** - macOS, Windows, and Linux
